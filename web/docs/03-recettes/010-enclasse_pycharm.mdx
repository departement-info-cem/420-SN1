# 🐙 Git avec PyCharm

Guide des différentes actions pour travailler avec un dépôt GitHub à l'aide de PyCharm (sans GitHub Desktop).

::::info-nt Commit and push
## 2️⃣ Commit and push
    <i>Action consistant à donner un nom significatif représentant les changements effectuées sur votre machine (**commit**),
    puis à les envoyer sur GitHub (**push**).</i><br/><br/>


    - Dans la barre de gauche, cliquer sur l’icône **« commit »** <img src={require('/img/icons/git_pycharm_icone_commit.png').default} style={{ border: '1px solid black' }} width="20"/>.
    - Cocher tous les fichiers ou modifications que vous souhaitez inclure dans le commit

    <p align="left">
        <img src={require('/img/recettes/git_pycharm_liste_changement.png').default} alt=" " style={{ verticalAlign: 'middle', marginLeft: '30px', border: '1px solid black' }} width="40%" />
    </p>


    - Dans la zone de texte située en dessous, écrire un **message clair décrivant les changements**

    <p align="left">
        <img src={require('/img/recettes/git_pycharm_description_commit.png').default} alt=" " style={{ verticalAlign: 'middle', marginLeft: '30px', border: '1px solid black' }} width="40%" />
    </p>

    - Cliquer sur **« commit and push... »**
    <p align="left">
        <img src={require('/img/recettes/git_pycharm_bouton_commit_and_push.png').default} alt=" " style={{ verticalAlign: 'middle', marginLeft: '30px', border: '1px solid black' }} width="30%" />
    </p>

    - Si demandé, saisir votre **courriel** et **nom d'utilisateur GitHub**, puis cliquer sur **« Set and Commit and Push »**
    - Appuyer sur le bouton **« push »** (en bleu)

    Vos changements devraient maintenant être envoyés sur GitHub (っ▀¯▀)つ
    :::warning
    Pensez à vérifier sur GitHub que votre commit est bien présent avant de quittez votre poste!
    :::
::::

:::info-nt Pull
## 3️⃣ Récupérer les derniers changements

    <i>Action consistant à mettre à jour la copie locale de votre projet en récupérant les dernières changements présents sur GitHub.</i><br/><br/>

    - Ouvrir votre projet dans PyCharm
    - Dans la barre de gauche, cliquer sur l’icône **« Git »** <img src={require('/img/icons/git_pycharm_icone_git.png').default} style={{ border: '1px solid black' }} width="20"/>
    - Sous la section **« Local »**, sélectionner la branche **« main »**
    - Cliquez sur l'icône **« Update Selected »** pour récupérer les derniers changements

    <p align="left">
        <img src={require('/img/recettes/git_pycharm_pull.png').default} alt=" " style={{ verticalAlign: 'middle', marginLeft: '30px', border: '1px solid black' }} width="30%" />
    </p>

    Votre projet PyCharm est maintenant à jour!
:::

:::tip-nt Questions fréquentes
## ❓ Question fréquente : je ne vois pas mes changements sur GitHub!

    Vous avez cliqué sur le bouton **« Commit »** au lieu de **« Commit and Push... »**. \
    Un **commit** sauvegarde vos changements **uniquement sur votre machine**. \
    Tant que vous n'avez pas fait un **push**, rien n'est envoyé sur GitHub!

    Pas de panique : vos changements sont déjà commités, il ne reste qu'à les **pousser (*push*)** vers GitHub.

    Voici comment faire :

    - Ouvrir votre projet dans **PyCharm**
    - Dans la barre de gauche, cliquer sur l'icône **« Git »** <img src={require('/img/icons/git_pycharm_icone_git.png').default} style={{ border: '1px solid black' }} width="20"/>
    - Sous la section **« Local »**, repérer la branche **« main »**.
    - Faire un **clic droit** sur la branche **« main »**, puis choisir **« Push... »**
    - Dans la fenêtre qui s'ouvre, vérifier vos changements, puis cliquer sur le bouton **« Push »** (en bleu)

    Vos changements devraient maintenant être visibles sur GitHub (っ▀¯▀)つ \
    Pensez à rafraîchir la page de votre dépôt sur GitHub pour le confirmer.
:::
