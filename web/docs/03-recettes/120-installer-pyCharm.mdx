# 🚀 Installer PyCharm

:::danger-nt
*PyCharm* est déjà installé sur les ordinateurs du cégep.\
Cette procédure est seulement pour installer *PyCharm* sur votre ordinateur personnel.
:::

*PyCharm* est le logiciel qui va nous servir pour coder en Python. \
Il est gratuit et est l'un des logiciels de référence dans l'industrie pour le langage Python.

Pour installer PyCharm à la maison :

1. Aller sur le site de [PyCharm](https://www.jetbrains.com/pycharm/download/).
2. Trouver la version *PyCharm Unified Product*.
3. Cliquer sur le lien de téléchargement (*Download*).
4. Double-cliquer sur le fichier "pycharm-2026.1.2.exe" dans vos téléchargements.
5. Suivre les instructions d'installation et cocher les options suivants :
    - *Create Desktop Shortcut / PyCharm*, afin de créer un raccourci de l'application sur votre bureau.
    - *Create Associations / .py*, afin d'associer le double-clic sur un fichier Python à l'application PyCharm.

## Tester l'installation de PyCharm

1. Lancer PyCharm.
2. [Créer un nouveau projet.](creer-projet)
3. Créer un nouveau fichier Python.
4. Écrire un petit programme et le rouler, par exemple :

```python
print("Hello world")
```

5. Exécuter le programme.

Vous devriez voir dans le bas de PyCharm sur fond noir, le message "Hello world" apparaitre. \
Si c'est bien le cas, votre installation est fonctionnelle.
