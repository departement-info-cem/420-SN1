# 📥 Créer un répertoire GitHub

:::info-nt

1. Allez sur [github.com](https://github.com) et connectez-vous à votre compte
2. Cliquez sur l'onglet **`Repositories`** en haut de la page
3. Cliquez sur le bouton vert **`New`** en haut à droite
4. Dans la section **`General`** :
   - Dans le champ **`Repository name`**, nommez votre dépôt en suivant les directives de votre enseignant.<br/>
     Par exemple, pour votre dépôt de code contenant les exercices du cours, le format attendu est :<br/><br/>`A26-GrXXXX-MATRICULE-PRENOM-NOM` *(ex : A26-Gr1010-123456-John-Doe)* où
     <br/>`XXXX` est votre numéro de groupe,
     <br/>`MATRICULE` votre matricule étudiant,
     <br/>`PRENOM` et `NOM` vos prénom et nom de famille.


   - Le champ **`Description`** est optionnel, mais il est recommandé d'y décrire au minimum le rôle du dépôt. <br/>Par exemple : *"Exercices réalisés dans le cadre du cours 420-SN1"*.<br/><br/>
5. Dans la section **`Configuration`** :
   - **`Choose visibility`** : sélectionnez **`Private`** \
        Votre dépôt ne sera visible que par vous et les collaborateurs que vous invitez.
   - **`Add README`** : activez le bouton pour le mettre à **`ON`** \
        Le contenu de ce fichier s'affiche sur la page d'accueil de votre dépôt et sert de carte de visite pour votre projet. \
        GitHub créera automatiquement un fichier `README.md` en reprenant la description que vous avez saisie plus haut.


   - **`Add .gitignore`** : sélectionnez **`Python`** \
        Cela exclut automatiquement les fichiers inutiles de votre dépôt de code.
   - **`Add license`** : laissez à **`No license`**
6. Cliquez sur **`Create repository`**

:::

# 👥 Ajouter un collaborateur à votre dépôt

:::info-nt

Une fois votre dépôt créé, vous devez y inviter votre enseignant comme collaborateur pour qu'il puisse consulter votre code. Vous pouvez trouver le nom de votre enseignant sur votre horaire de cours reçu en début de session.

1. Sur la page de votre dépôt, cliquez sur l'onglet **`Settings`**
2. Dans le menu de gauche, cliquez sur **`Collaborators`**
3. Cliquez sur le bouton **`Add people`**
4. Entrez le nom d'utilisateur GitHub de votre enseignant, puis cliquez sur **`Add [nom] to this repository`**
5. Votre enseignant recevra une invitation par courriel qu'il devra accepter
