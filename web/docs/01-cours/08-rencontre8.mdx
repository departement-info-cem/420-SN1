        <details>
        <summary>📁 Exercice - Calcul de la note finale du cours, avec la double sanction!</summary>

        Créez un fichier `r08_note_finale_double_sanction.py` pour votre réponse.

        Reprenez votre code de l'exercice **`r02_note_finale.py`** et améliorez-le en y appliquant la **règle de la double sanction**.

        #### 📋 Rappel — la règle de la double sanction

        Pour réussir le cours, un étudiant doit obtenir un minimum de **60 %** dans **chacune** des deux catégories suivantes :

        - Les évaluations **supervisées** : examens et contrôles.
        - Les évaluations **non supervisées** : travaux pratiques (TP) et devoirs.

        Si l'une ou l'autre de ces moyennes est **sous 60 %**, la note affichée au bulletin n'est pas la note finale pondérée, mais bien **la plus basse** des deux moyennes.

        #### 🛠️ INSTRUCTIONS

        1. **Repartez du code de `r02_note_finale.py`** (les 4 listes parallèles et le calcul de la note finale).

        2. Ajoutez une cinquième liste indiquant la **catégorie** de chaque évaluation :
            ```python
            categories = ["non_supervisee", "non_supervisee", "supervisee", "supervisee", "supervisee"]
            ```

        3. **Créez une fonction** `calculer_moyenne_categorie(notes, sur, ponderations, categories, categorie_cible)` qui :
            - parcourt les listes par index,
            - additionne les notes **pondérées** des évaluations appartenant à `categorie_cible`,
            - retourne cette somme **ramenée sur 100** (divisée par la somme des pondérations de cette catégorie).

        4. **Créez une fonction** `appliquer_double_sanction(note_finale, moyenne_supervisees, moyenne_non_supervisees)` qui :
            - retourne la note finale **telle quelle** si les deux moyennes sont ≥ 60 %,
            - retourne **la plus basse** des deux moyennes sinon.

        5. **Affichez un résumé clair** incluant :
            - le détail de chaque évaluation (comme dans R02),
            - la note finale **avant** double sanction,
            - la moyenne des évaluations supervisées,
            - la moyenne des évaluations non supervisées,
            - la note finale **après** double sanction,
            - un message indiquant si la double sanction a été appliquée ou non.

        Validez votre programme avec les **3 cas** ci-dessous.

        ✅ **Cas 1 : double sanction non déclenchée** (`notes = [8, 9, 90, 95, 9]`)
        ```
        Étudiant : Nicola Tesla

        TP1        (15%) :  8 / 10
        TP2        (25%) :  9 / 10
        Examen 1   (20%) : 90 / 100
        Examen 2   (30%) : 95 / 100
        Contrôles  (10%) :  9 / 10

        Note finale (avant double sanction)  : 90.00 / 100
        Moyenne évaluations supervisées      : 92.50 / 100
        Moyenne évaluations non supervisées  : 86.25 / 100

        ✅ Double sanction non déclenchée.
        Note finale : 90.00 / 100
        ```

        ✅ **Cas 2 : double sanction déclenchée par les supervisées** (`notes = [9, 9, 40, 45, 4]`)
        ```
        Étudiant : Nicola Tesla

        TP1        (15%) :  9 / 10
        TP2        (25%) :  9 / 10
        Examen 1   (20%) : 40 / 100
        Examen 2   (30%) : 45 / 100
        Contrôles  (10%) :  4 / 10

        Note finale (avant double sanction)  : 61.50 / 100
        Moyenne évaluations supervisées      : 42.50 / 100
        Moyenne évaluations non supervisées  : 90.00 / 100

        ⚠️ Double sanction déclenchée.
        Note finale : 42.50 / 100
        ```

        ✅ **Cas 3 : double sanction déclenchée par les non supervisées** (`notes = [2, 2, 90, 90, 9]`)
        ```
        Étudiant : Nicola Tesla

        TP1        (15%) :  2 / 10
        TP2        (25%) :  2 / 10
        Examen 1   (20%) : 90 / 100
        Examen 2   (30%) : 90 / 100
        Contrôles  (10%) :  9 / 10

        Note finale (avant double sanction)  : 62.00 / 100
        Moyenne évaluations supervisées      : 90.00 / 100
        Moyenne évaluations non supervisées  : 20.00 / 100

        ⚠️ Double sanction déclenchée.
        Note finale : 20.00 / 100
        ```

        :::note Notions visées
        Structures conditionnelles (`if`/`elif`/`else`), définition de fonctions, paramètres avec valeur par défaut, `return`, listes parallèles, accès par index
        :::
