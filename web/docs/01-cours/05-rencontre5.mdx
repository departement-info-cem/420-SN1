    <TabItem value="while" label="while" default>

        :::info Introduction aux boucles

        La boucle apporte à la programmation une puissance considérable.<br />
        Elle permet de répéter des actions, de parcourir des listes, de traiter des données
        de manière efficace.<br />
        Elle est un des piliers de la programmation avec les conditions et les fonctions.

        En gros, sa structure est la suivante :
        ```pseudo showLineNumbers
        Tant que (une certaine condition est vraie) :
            # instructions à répéter
            # instructions à répéter
            # instructions à répéter

        ```
        Si on exécutait ce pseudo-code, on répéterait l'exécution des lignes 2 à 4 tant que la condition de la ligne 1 est vraie.<br />
        Dès que la condition devient fausse, on sort de la boucle et on continue l'exécution du programme à la ligne 5.

        En Python, on utilise le mot-clé `while` pour créer une boucle de ce type.<br />
        Voici un exemple simple :
        ```python showLineNumbers
        compteur = 0
        while compteur < 5:
            print("Compteur =", compteur)
            compteur = compteur + 1  # Incrémente le compteur de 1
        print("Fin de la boucle")
        ```
        Dans cet exemple, la boucle `while` continue tant que la variable `compteur` est inférieure à 5, c'est-à-dire tant que l'expression `compteur < 5` est évaluée à `True`.<br />
        À chaque ***itération*** (tour de la boucle), elle affiche la valeur du compteur (ligne 3) et l'incrémente de 1 (ligne 4).<br />
        Quand `compteur` atteint 5, la condition (`compteur < 5`) devient fausse (`False`) et la boucle se termine.

        Exécutez ce code en mode débogage pour bien comprendre son fonctionnement.<br />
        Mettez un point d'arrêt sur la ligne 1 et avancez une ligne à la fois pour suivre l'évolution de la variable `compteur`.

        :::

        ---

        :::warning Attention aux boucles infinies
        Il est crucial de s'assurer que la condition de la boucle deviendra fausse à un moment donné.<br />
        Sinon, la boucle continuera indéfiniment, ce qui peut faire planter votre programme.<br />
        Dans l'exemple ci-dessus, si on oublie d'incrémenter `compteur`, la condition `compteur < 5` restera toujours vraie et la boucle ne s'arrêtera jamais.
        :::

        ---

        :::tip affectation raccourcie
        Pour incrémenter une variable, on peut utiliser l'opérateur d'affectation raccourcie `+=`.<br />
        Par exemple, au lieu d'écrire `compteur = compteur + 1`, on peut écrire `compteur += 1`.<br />
        Cela fonctionne de la même manière pour les autres opérations mathématiques avec les opérateurs `-=`, `*=`, `/=`, etc.

        Voici le même exemple avec l'opérateur `+=` :
        ```python showLineNumbers
        compteur = 0
        while compteur < 5:
            print("Compteur =", compteur)
            compteur += 1  # Incrémente le compteur de 1
        print("Fin de la boucle")
        ```
        :::

        ---

        :::info Autre exemple
        Voici un autre exemple qui utilise une boucle `while` pour calculer la somme des nombres de 1 à 10 :
        ```python showLineNumbers
        somme = 0
        nombre = 1
        while nombre <= 10:
            somme += nombre  # Ajoute le nombre courant à la somme
            nombre += 1      # Passe au nombre suivant
        print("La somme des nombres de 1 à 10 est :", somme)
        ```
        Dans cet exemple, la boucle continue tant que `nombre` est inférieur ou égal à 10 (`nombre <= 10`).<br />
        À chaque itération, elle ajoute la valeur de `nombre` à `somme` et incrémente `nombre` de 1.<br />
        Quand `nombre` devient 11, la condition `nombre <= 10` devient fausse et la boucle se termine.<br />
        Le programme affiche alors la somme totale des nombres de 1 à 10.

        Exécutez-le en mode débogage pour suivre l'évolution des variables `somme` et `nombre` à chaque itération!
        :::

        :::info Opérateurs d'affectation

        Nous avons déjà vu l'opérateur d'affectation `=` qui permet d'affecter une valeur à une variable.<br />
        Par exemple :
        ```python
        x = 5
        y = 10
        ```

        Python propose également des opérateurs d'affectation composés qui permettent de faire des
        opérations mathématiques et de réaffecter le résultat à la même variable en une seule instruction.

        Voici les opérateurs d'affectation composés les plus courants :

        | Opérateur | Description                          | Exemple               | Équivalent à          |
        |-----------|--------------------------------------|-----------------------|-----------------------|
        | <C>+=</C> | Addition et affectation | <C>a += b</C> | <C>a = a + b</C> |
        | <C>-=</C> | Soustraction et affectation | <C>a -= b</C> | <C>a = a - b</C> |
        | <C>*=</C> | Multiplication et affectation | <C>a *= b</C> | <C>a = a * b</C> |
        | <C>/=</C> | Division et affectation | <C>a /= b</C> | <C>a = a / b</C> |
        | <C>%=</C> | Modulo et affectation | <C>a %= b</C> | <C>a = a % b</C> |
        | <C>//=</C> | Division entière et affectation | <C>a //= b</C> | <C>a = a // b</C> |
        | <C>**=</C> | Exponentiation et affectation | <C>a **= b</C> | <C>a = a ** b</C> |

        Ces opérateurs sont très utiles pour simplifier le code et rendre les opérations plus concises.

        :::

        ---

        :::tip Exemple d'utilisation
        ```python
        compteur = 0
        compteur += 1  # équivalent à compteur = compteur + 1
        compteur *= 2  # équivalent à compteur = compteur * 2
        print(compteur)  # Affiche 2
        ```
        :::

    <TabItem value="for" label="range / for">

        :::tip La fonction `range()`
        `range()` génère une séquence de nombres. Il existe 3 façons de l'appeler selon le nombre de paramètres :

        - `range(stop)` : part de `0`, avance de `1` en `1`, s'arrête avant `stop`
        - `range(start, stop)` : part de `start`, avance de `1` en `1`, s'arrête avant `stop`
        - `range(start, stop, step)` : part de `start`, avance de `step` en `step`, s'arrête avant `stop`

        | Appel             | Produit les nombres    |
        |-------------------|------------------------|
        | <C><B>range</B>(<N>5</N>)</C> | <C><N>0</N>, <N>1</N>, <N>2</N>, <N>3</N>, <N>4</N></C> |
        | <C><B>range</B>(<N>1</N>, <N>5</N>)</C> | <C><N>1</N>, <N>2</N>, <N>3</N>, <N>4</N></C> |
        | <C><B>range</B>(<N>0</N>, <N>11</N>, <N>2</N>)</C> | <C><N>0</N>, <N>2</N>, <N>4</N>, <N>6</N>, <N>8</N>, <N>10</N></C> |
        | <C><B>range</B>(<N>5</N>, <N>0</N>, -<N>1</N>)</C> | <C><N>5</N>, <N>4</N>, <N>3</N>, <N>2</N>, <N>1</N></C> |

        ⚠️ `range()` n'est pas exactement une liste en Python, mais elle se comporte de la même façon lorsqu'on la parcourt.
        :::

        ---

        ::::info Boucle `for` avec `range()`
        La boucle `for` parcourt une séquence de valeurs **une par une**, et exécute les instructions du bloc à chaque itération (à chaque tour de boucle).

        Sa structure est la suivante :
        ```python showLineNumbers
        for i in range(start, stop, step):
            # instructions répétées à chaque itération
        ```

        À chaque itération de boucle, `i` prend automatiquement la valeur suivante dans la séquence produite par `range()`.

        :::tip À noter
        La boucle `for` est un cas particulier de `while`. Elle est simplement plus pratique quand on sait d'avance combien de tours on veut faire.
        :::

        Voici quatre exemples, lisez d'abord le commentaire, puis le code :
        ```python showLineNumbers
        # Exemple 1 : i vaut 0, puis 1, puis 2, puis 3, puis 4 → 5 tours
        for i in range(5):
            print(i)
        ```
        ```python showLineNumbers
        # Exemple 2 : i commence à 1 et s'arrête avant 6 → affiche 1 à 5
        for i in range(1, 6):
            print(i)
        ```
        ```python showLineNumbers
        # Exemple 3 : i avance de 2 en 2 → affiche seulement les pairs
        for i in range(0, 11, 2):
            print(i)
        ```
        ```python showLineNumbers
        # Exemple 4 : step négatif → i descend de 5 jusqu'à 1
        for i in range(5, 0, -1):
            print(i)
        ```

        👉 Exécutez ces exemples en mode débogage, un à la fois.<br />
        Mettez un point d'arrêt sur le `for` et avancez ligne par ligne pour voir `i` changer de valeur à chaque itération.
        ::::

        ---

        :::info Parcourir une liste avec `for`, `range()` et `len()`
        Puisque `range()` produit une séquence d'index (`0`, `1`, `2`…), on peut l'utiliser pour accéder à chaque élément d'une liste tour par itération.

        `len(liste)` retourne le nombre d'éléments d'une liste. Donc `range(len(liste))` génère exactement les bons index pour parcourir toute la liste :

        ```python showLineNumbers
        consoles = ["PS5", "Xbox Series X", "Nintendo Switch 2"]
        nb_elements = len(consoles)       # nb_elements vaut 3
        for i in range(nb_elements):      # i prendra les valeurs 0, 1, 2
            print(consoles[i])            # accède à l'élément à la position i

        # On aurait pu écrire directement :
        # for i in range(len(consoles)):
        ```

        Au 1er tour : `i` vaut `0` → `consoles[0]` → `"PS5"` \
        Au 2e tour &nbsp;: `i` vaut `1` → `consoles[1]` → `"Xbox Series X"` \
        Au 3e tour &nbsp;: `i` vaut `2` → `consoles[2]` → `"Nintendo Switch 2"`

        👉 Exécutez ce code en mode débogage et observez `i` et `consoles[i]` évoluer ensemble à chaque itération.
        :::

        ---

        :::tip Quand on n'a pas besoin de `i` : utiliser `_`
        Parfois, on veut simplement **répéter une action** un certain nombre de fois, sans jamais utiliser la valeur de `i` à l'intérieur de la boucle.

        Par exemple, afficher 5 fois le même message :
        ```python showLineNumbers
        for i in range(5):
            print("Bienvenue!")  # i n'est pas utilisé
        ```

        Dans ce cas, nommer la variable `i` est trompeur : ça laisse croire qu'on va s'en servir, alors qu'on ne l'utilise pas. \
        La convention en Python est d'utiliser `_` (un tiret bas) pour signaler explicitement que la variable est intentionnellement ignorée :

        ```python showLineNumbers
        for _ in range(5):
            print("Bienvenue!")  # on répète 5 fois, la valeur du tour ne nous intéresse pas
        ```

        Le résultat est identique, mais `_` communique clairement l'intention : *"je n'ai pas besoin de savoir à quel tour je suis, je veux juste répéter."*

        👉 Règle simple : si on utilise la variable de boucle dans le bloc (`consoles[i]`, `i * 2`, etc.), on la nomme `i`. Si on ne l'utilise jamais, on utilise `_`.
        :::

        :::info Structures combinées

        Il est possible de combiner les différentes structures de contrôle pour créer des programmes plus complexes et puissants.<br />
        Par exemple, on peut imbriquer des boucles à l'intérieur d'autres boucles, ou utiliser des conditions à l'intérieur de boucles.

        Voici un exemple simple qui combine une boucle `for` avec une condition `if` pour afficher les nombres pairs de 0 à 10 :
        ```python showLineNumbers
        for i in range(11):
            if i % 2 == 0:
                print(i)
        ```
        Dans cet exemple, la boucle `for` itère sur les nombres de 0 à 10, et la condition `if` vérifie si le nombre est pair (c'est-à-dire si le reste de la division par 2 est égal à 0).<br />
        Si la condition est vraie, le nombre est affiché.

        Exécutez ce code en mode débogage pour bien comprendre son fonctionnement.<br />
        Mettez un point d'arrêt sur la ligne 1 et avancez une ligne à la fois pour suivre l'évolution de la variable `i` et l'évaluation de la condition.

        :::

        ---

        :::info Exemple avec une liste
        On peut combiner `for` et `if` pour traiter les éléments d'une liste selon une condition. \
        Voici un exemple qui parcourt une liste de températures et affiche un message différent selon que la température indique une fièvre ou non :
        ```python showLineNumbers
        temperatures = [36.5, 38.2, 37.1, 39.8, 36.9]
        for i in range(len(temperatures)):
            if temperatures[i] > 37.5:
                print(f"{temperatures[i]}°C : fièvre!")
            else:
                print(f"{temperatures[i]}°C : normale")
        ```
        À chaque itération, `i` prend la valeur de l'index courant et `temperatures[i]` accède à la température correspondante dans la liste. \
        La condition `if` détermine ensuite quel message afficher pour cette température.

        Exécutez ce code en mode débogage pour suivre l'évolution de `i` et de `temperatures[i]` à chaque itération.
        :::

    <TabItem value="break" label="break / continue / pass">

        :::info Instructions `break`, `continue` et `pass`

        Parfois, on peut vouloir interrompre une boucle avant qu'elle ne se termine naturellement, ou sauter certaines itérations.<br />
        Python fournit trois instructions spéciales pour gérer ces situations : `break`, `continue` et `pass`.

        :::

        ---

        :::info `break`
        Cette instruction permet de sortir immédiatement d'une boucle, qu'elle soit `for` ou `while`.<br />
        Lorsqu'un `break` est rencontré, l'exécution du programme continue à la ligne suivant la boucle.
        ```python showLineNumbers
        for i in range(10):
            if i == 5:
                break  # Sort de la boucle lorsque i vaut 5
            print(i)
        print("Fin de la boucle")
        ```
        Dans cet exemple, la boucle s'arrête dès que `i` atteint 5.

        :::

        ---

        :::info `continue`
        Cette instruction permet de sauter le reste du code dans la boucle pour l'itération en cours et de passer directement à l'itération suivante.
        ```python showLineNumbers
        for i in range(10):
            if i % 2 == 0:
                continue  # Saute les nombres pairs
            print(i)
        ```
        Ici, seuls les nombres impairs sont affichés, car les nombres pairs sont ignorés grâce à `continue`.

        :::

        ---

        :::info `pass`
        Cette instruction est un "no-op" (no operation), c'est-à-dire qu'elle ne fait rien.<br />
        Elle est souvent utilisée comme un espace réservé dans des blocs de code où une instruction est requise syntaxiquement mais où aucune action n'est nécessaire.
        ```python showLineNumbers
        for i in range(5):
            if i < 3:
                pass  # Ne fait rien pour i < 3
            else:
                print(i)
        ```
        Dans cet exemple, rien ne se passe pour les valeurs de `i` inférieures à 3, mais pour les autres valeurs, elles sont affichées.

        :::

        ---

        :::tip Pour bien comprendre
        Exécutez ces exemples en mode débogage pour bien comprendre comment chaque instruction affecte le flux d'exécution de la boucle.<br />
        Mettez des points d'arrêt aux endroits stratégiques et avancez ligne par ligne pour observer le comportement du programme.
        :::

    <TabItem value="DRY" label="DRY">

        :::info Éviter la répétition : les boucles et les listes
        En programmation, on suit souvent le principe **DRY** (*Don't Repeat Yourself*).<br />
        Si vous remarquez que vous copiez-collez un bloc d'instructions en changeant seulement quelques valeurs, c'est le signe qu'il faut utiliser une **boucle**!
        :::

        ---

        :::danger ⚠️ Le code redondant
        Voici un programme qui affiche les informations de trois étudiants. Le bloc d'instructions est répété trois fois, seules les valeurs changent :
        ```python showLineNumbers
        print("Étudiant #1")
        print("Nom : Félix")
        print("Note : 72")
        print()

        print("Étudiant #2")
        print("Nom : Dayron")
        print("Note : 91")
        print()

        print("Étudiant #3")
        print("Nom : Léa")
        print("Note : 58")
        print()
        ```
        Ce code fonctionne, mais il est difficile à maintenir : ajouter un 4e étudiant oblige à copier-coller un nouveau bloc. Et si le format d'affichage change, il faut modifier trois endroits!
        :::

        ---

        :::note Étape 1 : Identifier ce qui change
        En observant les trois blocs, on remarque que seules trois choses varient d'un bloc à l'autre :
        - Le **numéro** de l'étudiant (`#1`, `#2`, `#3`)
        - Le **nom** (`Félix`, `Dayron`, `Léa`)
        - La **note** (`72`, `91`, `58`)

        Le numéro correspond simplement à la position dans la liste (index `0`, `1`, `2`), donc on peut le calculer automatiquement avec `i + 1`.<br />
        On regroupe les noms et les notes dans des **listes parallèles** :
        ```python showLineNumbers
        noms  = ["Félix", "Dayron", "Léa"]
        notes = [72, 91, 58]
        ```
        :::

        ---

        :::tip ✅ Étape 2 : Remplacer la répétition par une boucle
        Maintenant qu'on a les données dans des listes, une seule boucle suffit pour traiter tous les étudiants :
        ```python showLineNumbers
        noms  = ["Félix", "Dayron", "Léa"]
        notes = [72, 91, 58]

        nb_elements = len(noms)
        for i in range(nb_elements):
            print(f"Étudiant #{i + 1}")
            print(f"Nom : {noms[i]}")
            print(f"Note : {notes[i]}")
            print()
        ```
        Le résultat affiché est **identique** à la version redondante, mais le code est bien plus court et plus facile à maintenir.
        :::

        ---

        :::info L'avantage clé : la flexibilité
        Pour ajouter un 4e étudiant, on n'ajoute qu'**un élément dans chaque liste** et la boucle s'adapte automatiquement grâce à `len()` :
        ```python showLineNumbers
        noms  = ["Félix", "Dayron", "Léa", "Tristan"]
        notes = [72, 91, 58, 83]

        nb_elements = len(noms)
        for i in range(nb_elements):
            print(f"Étudiant #{i + 1}")
            print(f"Nom : {noms[i]}")
            print(f"Note : {notes[i]}")
            print()
        ```
        👉 Le corps de la boucle n'a pas changé d'une seule ligne. C'est la puissance de combiner les listes et les boucles!
        :::

    <TabItem value="forfor" label="Boucles imbriquées">

        :::info Boucles imbriquées

        Il est possible de placer une boucle à l'intérieur d'une autre boucle.<br />
        On parle alors de boucles imbriquées.

        Voici un exemple simple avec deux boucles `for` imbriquées pour afficher une table de multiplication :
        ```python showLineNumbers
        for i in range(1, 6):           # Boucle extérieure pour les lignes
            for j in range(1, 6):       # Boucle intérieure pour les colonnes
                print(i * j, end='\t')  # Affiche le produit avec une tabulation
            print()                     # Nouvelle ligne après chaque ligne de la table
        ```
        Dans cet exemple, la boucle extérieure itère sur les valeurs de `i` de 1 à 5, et pour chaque valeur de `i`, la boucle intérieure itère sur les valeurs de `j` de 1 à 5.<br />
        Le produit `i * j` est affiché dans une grille formatée.

        Exécutez ce code en mode débogage pour bien comprendre son fonctionnement.<br />
        Mettez un point d'arrêt sur la ligne 1 et avancez une ligne à la fois pour suivre l'évolution des variables `i` et `j`.

        :::

        ---

        :::tip Exemple avec une boucle `while`
        Voici un exemple similaire utilisant des boucles `while` imbriquées pour afficher un triangle d'étoiles :
        ```python showLineNumbers
        lignes = 5
        i = 1
        while i <= lignes:           # Boucle extérieure pour les lignes
            j = 1
            while j <= i:            # Boucle intérieure pour les étoiles dans chaque ligne
                print('*', end=' ')  # Afficher une étoile sans générer un saut de ligne
                j += 1
            print()                  # Nouvelle ligne après chaque ligne d'étoiles
            i += 1
        ```
        Dans cet exemple, la boucle extérieure contrôle le nombre de lignes, tandis que la boucle intérieure contrôle le nombre d'étoiles affichées dans chaque ligne.

        Exécutez ce code en mode débogage pour bien comprendre son fonctionnement.<br />
        Mettez un point d'arrêt sur la ligne 2 et avancez une ligne à la fois pour suivre l'évolution des variables `i` et `j`.

        :::

        ---

        :::note Exemple
        Voici une fonction qui prend comme paramètres une largeur et une hauteur, et qui affiche un rectangle de `#` de ces dimensions :
        ```python showLineNumbers
        def afficher_rectangle(largeur, hauteur):
            for i in range(hauteur):          # Boucle pour chaque ligne
                for j in range(largeur):      # Boucle pour chaque colonne
                    print('#', end='')        # Affiche un '#' sans saut de ligne
                print()                       # Nouvelle ligne après chaque ligne de '#'
        afficher_rectangle(5, 3)
        ```
        Exécutez ce code en mode débogage pour bien comprendre son fonctionnement.<br />
        Il devrait afficher :
        ```
        #####
        #####
        #####
        ```

        Voici  maintenant une version qui utilise la concaténation de chaînes de caractères (str) pour construire chaque ligne avant de l'afficher :
        ```python showLineNumbers
        def afficher_rectangle(largeur, hauteur):
            for i in range(hauteur):          # Boucle pour chaque ligne
                ligne = ''                    # Initialise une chaîne vide pour la ligne
                for j in range(largeur):      # Boucle pour chaque colonne
                    ligne += '#'              # Concatène un '#' à la chaîne de la ligne
                print(ligne)                  # Affiche la ligne complète
        afficher_rectangle(5, 3)
        ```

        Vous remarquerez que le résultat est le même, mais la méthode est différente.<br />
        Entre autres, cela évite d'utiliser `print()` à chaque itération de la boucle intérieure, ce qui peut être plus efficace pour de grandes dimensions.

        Exécutez ce code en mode débogage pour bien voir la `str` `ligne` s'allonger à chaque itération de la boucle intérieure.

        :::

        ::::info-nt

        ### La fonction `input()`
        La fonction `input()` permet de demander à l'utilisateur de saisir une entrée.<br />
        Elle peut être utilisée avec ou sans paramètre.<br />
        - Sans paramètre, elle attend une saisie de l'utilisateur.
        - Avec un paramètre, elle affiche un message pour guider l'utilisateur.

        Par exemple (essayez ce code dans un script Python) :
        ```python showLineNumbers
        # Exemple sans paramètre
        saisie_utilisateur = input()  # attend une saisie de l’utilisateur
        print(saisie_utilisateur)

        # Exemple avec un paramètre
        prenom = input("Veuillez saisir votre prénom : ")
        print(prenom)
        ```

        :::warning La fonction input met le programme sur pause!
         Lorsque la fonction `input(...)` est exécutée, le programme se met en pause. \
         Le programme attend que l’utilisateur.rice écrive quelque chose dans la console et qu’il ou elle appuie sur la touche ENTRÉE. \
         C’est donc **normal** que l’instruction suivante ne soit pas exécutée tant que vous n’avez pas saisi quelque chose dans la console!
        :::
        La fonction `input(...)` retourne **TOUJOURS** une chaîne de caractères (`str`), même si l’utilisateur saisit un nombre. \
        Pour utiliser la valeur dans un calcul, il faut la convertir avec `int()` ou `float()`.

        Voici deux exemples concrets :

        :::tip Demander un nombre entier ou décimal
        ```python showLineNumbers
        # Demander un nombre entier (int)
        age = int(input("Entrez votre âge : "))
        print(f"Dans 10 ans, vous aurez {age + 10} ans.")

        # Demander un nombre décimal (float)
        taille = float(input("Entrez votre taille en mètres : "))
        print(f"Votre taille en centimètres est {taille * 100} cm.")
        ```

        `int()` convertit la chaîne en nombre entier et `float()` en nombre décimal.<br />
        Sans cette conversion, Python traiterait la saisie comme du texte et l’addition ou la multiplication ne fonctionnerait pas comme prévu.
        :::
