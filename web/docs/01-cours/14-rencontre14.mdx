        ### 🔒 Volet 1 : Partie papier (sans ordinateur, sans documentation)
        📄 3 questions :

            🔹Tracer un graphique correspondant au code présenté (2 points)  \
            &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- dessiner des graphiques de difficulté similaire à ce qui est dans R02, R09 et R10 (graphique sans Pandas)

            🔹Trouver le contenu affiché par les instructions `print()` du code présenté (2 points) \
            &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- pas de trace à l'examen, vous devez écrire le contenu affiché par le programme \
            &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- plus d'une trentaine d'exercices de ce genre vous sont proposés dans l'onglet suivant \
            &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- ces exercices révisent beaucoup d'éléments du cours, à consommer sans modération!

            🔹Écrire du code pour résoudre un problème (2 points) \
            &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- niveau de difficulté similaire aux exercices de ce genre dans R11 \
            &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- quelques exercices supplémentaires vous sont proposés dans l'onglet suivant \
            &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- aucune utilisation de modules, de bibliothèques (Matplotlib, Pandas, NumPy, etc) ou de fichiers \
            &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- il est nécessaire de connaître le reste des notions par cœur (ex : condition, boucle, liste, dictionnaire, tuple)

        ✅ Une fois terminé :

            🔹Remettez votre copie à l’enseignant.e. \
            🔹L’enseignant déverrouillera votre ordinateur pour accéder au volet 2.

        ### 💻 Volet 2 : Partie sur ordinateur

        🧠 3 questions :

        🔸 Graphique (2 points) : *Réaliser un graphique. Les données devront être adaptées pour être utilisables par Matplotlib (graphique sans Pandas).*   \
        🔸 Fichier CSV (2 points) : *Analyser un fichier CSV afin de répondre à différentes questions d'analyse à l'aide de Pandas.*  \
        🔸 Synthèse (2 points) : *Intégration d'un large ensemble de notions pour résoudre un problème.*

        🕵️‍♂️ Vous devez **identifier par vous-même** les notions nécessaires à chaque question.
