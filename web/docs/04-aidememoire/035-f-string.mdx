# 🆎 String & F-string

Cet aide-mémoire couvre deux aspects des chaînes de caractères : les **méthodes** disponibles sur toute chaîne, et les options de **formatage** propres aux *f-string*.

---

## 🔹 Méthodes utiles sur les chaînes

Ces méthodes s'appliquent directement sur n'importe quelle chaîne de caractères.

| Méthode | Description | Exemple |
|-|-|-|
| <C><F>upper</F>()</C> | Convertit en majuscules | <C><S>"mario"</S>.<F>upper</F>()</C> → <C><S>"MARIO"</S></C> |
| <C><F>lower</F>()</C> | Convertit en minuscules | <C><S>"MARIO"</S>.<F>lower</F>()</C> → <C><S>"mario"</S></C> |
| <C><F>capitalize</F>()</C> | Première lettre en majuscule | <C><S>"mario"</S>.<F>capitalize</F>()</C> → <C><S>"Mario"</S></C> |
| <C><F>strip</F>()</C> | Supprime les espaces aux extrémités | <C><S>"  mario  "</S>.<F>strip</F>()</C> → <C><S>"mario"</S></C> |
| <C><F>replace</F>(ancien, nouveau)</C> | Remplace une sous-chaîne | <C><S>"mario"</S>.<F>replace</F>(<S>"mario"</S>, <S>"link"</S>)</C> → <C><S>"link"</S></C> |
| <C><F>split</F>(séparateur)</C> | Découpe en liste selon un séparateur | <C><S>"mario,link"</S>.<F>split</F>(<S>","</S>)</C> → <C>[<S>"mario"</S>, <S>"link"</S>]</C> |
| <C><S>" "</S>.<F>join</F>(liste)</C> | Fusionne une liste en chaîne | <C><S>" "</S>.<F>join</F>([<S>"mario"</S>, <S>"link"</S>])</C> → <C><S>"mario link"</S></C> |

---

## 🔹 Formatage avec les f-string

Les options ci-dessous s'utilisent à l'intérieur d'une *f-string*, immédiatement avant l'accolade fermante.

```python
# le résultat est arrondi à 2 décimales :
print(f"Le résultat de {nombre1} x {nombre2} est : {nombre1 * nombre2:.2f}.")
```

### Formatage des nombres

| Syntaxe | Effet | Exemple |
|-|-|-|
| <C><S>:.2f</S></C> | Arrondi à 2 décimales, complété par des 0 | <C><N>3.14159</N></C> → <C><N>3.14</N></C> / <C><N>3.5</N></C> → <C><N>3.50</N></C> |
| <C><S>:.0f</S></C> | Arrondi sans décimale | <C><N>3.99</N></C> → <C><N>4</N></C> |
| <C><S>:.3e</S></C> | Notation scientifique après la 3ème décimale | <C><N>123456</N></C> → <C><N>1.235e+05</N></C> |
| <C><S>:,</S></C> | Spécifier le séparateur de milliers | <C><N>1000000</N></C> → <C><N>1,000,000</N></C> |
| <C><S>:05d</S></C> | Entier sur 5 chiffres, complété par des 0 | <C><N>42</N></C> → <C><N>00042</N></C> |
| <C><S>:+.2f</S></C> | Affiche le signe + ou - et conserve 2 décimales | <C><N>3.5</N></C> → <C><N>+3.50</N></C> / <C>-<N>2</N></C> → <C>-<N>2.00</N></C> |

### Formatage des chaînes de caractères

| Syntaxe | Effet | Exemple |
|-|-|-|
| <C><S>:&lt;10</S></C> | Aligné à gauche sur 10 caractères | <C><S>"Mots"</S></C> → <C><S>Mots{'      '}</S></C> |
| <C><S>:&gt;10</S></C> | Aligné à droite sur 10 caractères | <C><S>"Mots"</S></C> → <C><S>{'      '}Mots</S></C> |
| <C><S>:^10</S></C> | Centré sur 10 caractères | <C><S>"Mots"</S></C> → <C><S>{'   '}Mots{'   '}</S></C> |
