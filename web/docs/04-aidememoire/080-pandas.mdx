# 🐼 Pandas

**Pandas** est une bibliothèque Python conçue pour charger, explorer, transformer et analyser des données tabulaires (comme des fichiers Excel ou CSV) en quelques lignes de code. Elle est indispensable dès qu'on doit manipuler des centaines ou des milliers de données.

## 🔹 Importation
