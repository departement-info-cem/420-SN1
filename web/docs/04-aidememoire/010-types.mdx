# 🍱 Types de données

Voici les principaux types et structures de données utiles pour débuter en Python.

## Types de base

| Type      | Description            | Exemple            |
|-----------|------------------------|--------------------|
| **int**   | Nombre entier           | `x = 42`           |
| **float** | Nombre décimal          | `y = 3.14`         |
| **str**   | Chaîne de caractères    | `nom = "Alice"`    |
| **bool**  | Booléen (vrai ou faux)  | `actif = True`     |

## Structures de données

| Type      | Description                              | Exemple                           |
|-----------|------------------------------------------|------------------------------------|
| **list**  | Liste ordonnée et modifiable             | `fruits = ["pomme", "banane", "cerise"]`    |
| **tuple** | Liste ordonnée et immuable               | `coordonnes = (10, 20)`                |
| **set**   | Ensemble non ordonné, sans doublons      | `couleurs = {"rouge", "vert", "bleu"}`    |
| **dict**  | Dictionnaire (paires clé-valeur)         | `ages = {"Alice": 30, "Bob": 25}`     |
