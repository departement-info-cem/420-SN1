# 🧮 Opérateurs

Voici un résumé des principaux opérateurs que vous utiliserez en Python.<br />
Les opérateurs permettent de faire des calculs, des comparaisons et des opérations logiques.

<Tabs queryString="onglet">

<TabItem value="arith" label="Arithmétiques" default>

| Opérateur | Description | Exemple (`a = 10`, `b = 3`) | Valeur résultante |
|---|---|---|---|
| <C>+</C> | Addition | <C>a + b</C> | <C><N>13</N></C> |
| <C>-</C> | Soustraction | <C>a - b</C> | <C><N>7</N></C> |
| <C>*</C> | Multiplication | <C>a * b</C> | <C><N>30</N></C> |
| <C>/</C> | Division (float) | <C>a / b</C> | <C><N>3.333...</N></C> |
| <C>//</C> | Division entière | <C>a // b</C> | <C><N>3</N></C> |
| <C>%</C> | Modulo (reste) | <C>a % b</C> | <C><N>1</N></C> |
| <C>**</C> | Puissance | <C>a ** b</C> | <C><N>1000</N></C> |

</TabItem>

<TabItem value="affect" label="Affectation">

| Opérateur | Description | Exemple |
|---|---|---|
| <C>=</C> | Affectation simple | <C>x = <N>5</N></C> |
| <C>+=</C> | Ajoute et assigne | <C>x += <N>2</N></C> → <C>x = x + <N>2</N></C> |
| <C>-=</C> | Soustrait et assigne | <C>x -= <N>3</N></C> |
| <C>*=</C> | Multiplie et assigne | <C>x *= <N>4</N></C> |
| <C>/=</C> | Divise et assigne (float) | <C>x /= <N>2</N></C> |
| <C>//=</C> | Division entière et assigne | <C>x //= <N>2</N></C> |
| <C>%=</C> | Modulo et assigne | <C>x %= <N>2</N></C> |
| <C>**=</C> | Puissance et assigne | <C>x **= <N>3</N></C> |

</TabItem>

<TabItem value="compar" label="Comparaison">

| Opérateur | Description | Exemple (`x = 6`, `y = 7`) | Résultat |
|---|---|---|---|
| <C>==</C> | Égal à | <C>x == y</C> | <C><K>False</K></C> |
| <C>!=</C> | Différent de | <C>x != y</C> | <C><K>True</K></C> |
| <C>&gt;</C> | Supérieur à | <C>x &gt; y</C> | <C><K>False</K></C> |
| <C>&lt;</C> | Inférieur à | <C>x &lt; y</C> | <C><K>True</K></C> |
| <C>&gt;=</C> | Supérieur ou égal à | <C>x &gt;= y</C> | <C><K>False</K></C> |
| <C>&lt;=</C> | Inférieur ou égal à | <C>x &lt;= y</C> | <C><K>True</K></C> |

</TabItem>

<TabItem value="logique" label="Logiques (booléens)">

| Opérateur | Description | Exemple | Résultat |
|---|---|---|---|
| <C><K>and</K></C> | ET logique | <C>a <K>and</K> b</C> | <C><K>True</K></C> si `a` et `b` sont **tous deux** <C><K>True</K></C>, sinon <C><K>False</K></C> |
| <C><K>or</K></C> | OU logique | <C>a <K>or</K> b</C> | <C><K>True</K></C> si **au moins un** de `a` ou `b` est <C><K>True</K></C>, sinon <C><K>False</K></C> |
| <C><K>not</K></C> | NON logique | <C><K>not</K> a</C> | <C><K>True</K></C> si `a` est <C><K>False</K></C>, sinon <C><K>False</K></C> |

</TabItem>

<TabItem value="autres" label="Autres utiles">

| Opérateur | Description | Exemple | Résultat |
|---|---|---|---|
| <C><K>in</K></C> | Appartenance (contenu dans) | <C><S>"a"</S> <K>in</K> <S>"chat"</S></C> | <C><K>True</K></C> |
| <C><K>not</K> <K>in</K></C> | Non-appartenance (pas dans) | <C><S>"b"</S> <K>not</K> <K>in</K> <S>"chat"</S></C> | <C><K>True</K></C> |
| <C><K>is</K></C> | Identité d'objet (même objet) | <C>x <K>is</K> y</C> | |
| <C><K>is</K> <K>not</K></C> | Objet différent | <C>x <K>is</K> <K>not</K> y</C> | |

</TabItem>

<TabItem value="priorite" label="Priorité des opérateurs">

| Priorité (haut → bas) | Opérateur(s) | Description |
|---|---|---|
| 1 | <C>()</C> | Parenthèses (groupement, appels de fonction, tuples, etc.) |
| 2 | <C>**</C> | Puissance |
| 3 | <C>*</C>, <C>/</C>, <C>//</C>, <C>%</C> | Multiplication, division, division entière, modulo |
| 4 | <C>+</C>, <C>-</C> | Addition, soustraction |
| 5 | <C>==</C>, <C>!=</C>, <C>&lt;</C>, <C>&lt;=</C>, <C>&gt;</C>, <C>&gt;=</C>, <C><K>in</K></C>, <C><K>not</K> <K>in</K></C>, <C><K>is</K></C>, <C><K>is</K> <K>not</K></C> | Comparaisons et tests d'appartenance/identité |
| 6 | <C><K>not</K></C> | Négation logique |
| 7 | <C><K>and</K></C> | ET logique |
| 8 | <C><K>or</K></C> | OU logique |
| 9 | <C>=</C>, <C>+=</C>, <C>-=</C>, <C>*=</C>, <C>/=</C>, <C>//=</C>, <C>%=</C> ... | Affectations |

Si 2 opérateurs ont la même priorité, l'ordre d'évaluation se fait de gauche à droite.<br />
Par exemple, <C>a - b + c</C> est évalué comme <C>(a - b) + c</C>.

</TabItem>

</Tabs>
