# 🔢 NumPy

**NumPy** est une bibliothèque Python conçue pour effectuer des calculs sur des tableaux entiers en une seule instruction, sans boucle, avec une vitesse bien supérieure!. Elle est la base de bibliothèques comme Pandas et Scikit-learn.
