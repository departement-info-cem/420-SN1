# 🛠️ Fonctions utiles

Voici les fonctions Python les plus fréquemment utilisées pour manipuler des données, obtenir des informations ou réaliser des conversions.

## 🔹 Fonctions de base

| Fonction | Description | Exemple |
|---|---|---|
| <C><B>print</B>()</C> | Affiche du texte ou des variables | <C><B>print</B>(<S>"Bonjour"</S>)</C> |
| <C><B>len</B>()</C> | Renvoie la longueur d'une séquence | <C><B>len</B>(<S>"Salut"</S>)</C> → <C><N>5</N></C> |
| <C><B>type</B>()</C> | Renvoie le type d'une variable | <C><B>type</B>(<N>42</N>)</C> → <C>&lt;class <S>'int'</S>&gt;</C> |

## 🔹 Fonctions de conversion

| Fonction | Description | Exemple |
|---|---|---|
| <C><B>int</B>()</C> | Conversion en entier | <C><B>int</B>(<S>"5"</S>)</C> → <C><N>5</N></C> |
| <C><B>float</B>()</C> | Conversion en nombre décimal | <C><B>float</B>(<S>"3.14"</S>)</C> → <C><N>3.14</N></C> |
| <C><B>str</B>()</C> | Conversion en chaîne de caractères | <C><B>str</B>(<N>42</N>)</C> → <C><S>"42"</S></C> |
| <C><B>bool</B>()</C> | Conversion en booléen | <C><B>bool</B>(<N>0</N>)</C> → <C><K>False</K></C> |

## 🔹 Fonctions de saisie

| Fonction | Description | Exemple |
|---|---|---|
| <C><B>input</B>()</C> | Demande une saisie textuelle | <C>nom = <B>input</B>(<S>"Votre nom ? "</S>)</C> |
| <C><B>int</B>(<B>input</B>())</C> | Saisie convertie en entier | <C>age = <B>int</B>(<B>input</B>(<S>"Votre âge ? "</S>))</C> |
| <C><B>float</B>(<B>input</B>())</C> | Saisie convertie en nombre décimal | <C>taille = <B>float</B>(<B>input</B>(<S>"Votre taille ? "</S>))</C> |

## 🔹 Fonctions diverses

| Fonction | Description | Exemple |
|---|---|---|
| <C><B>range</B>()</C> | Génère une séquence de nombres | <C><B>range</B>(<N>5</N>)</C> → <C><N>0</N>, <N>1</N>, <N>2</N>, <N>3</N>, <N>4</N></C> |
| <C><B>round</B>()</C> | Arrondit un nombre | <C><B>round</B>(<N>3.14159</N>, <N>2</N>)</C> → <C><N>3.14</N></C> |
| <C><B>abs</B>()</C> | Valeur absolue d'un nombre | <C><B>abs</B>(-<N>7</N>)</C> → <C><N>7</N></C> |
| <C><B>pow</B>(base, exp)</C> | Puissance (équivalent à <C>base ** exp</C>) | <C><B>pow</B>(<N>2</N>, <N>3</N>)</C> → <C><N>8</N></C> |
