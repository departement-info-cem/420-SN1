Ouvrir, lire et écrire un fichier texte avec <C><K>with</K> <B>open</B>(...)</C>. Le fichier est automatiquement fermé à la fin du bloc.

| Opération | Syntaxe | Description |
|---|---|---|
| Ouvrir en lecture | <C><K>with</K> <B>open</B>(<S>"f.txt"</S>, <S>"r"</S>, encoding=<S>"utf-8"</S>) <K>as</K> f:</C> | Lit le contenu existant |
| Ouvrir en écriture | <C><K>with</K> <B>open</B>(<S>"f.txt"</S>, <S>"w"</S>, encoding=<S>"utf-8"</S>) <K>as</K> f:</C> | Écrase le contenu existant |
| Ouvrir en ajout | <C><K>with</K> <B>open</B>(<S>"f.txt"</S>, <S>"a"</S>, encoding=<S>"utf-8"</S>) <K>as</K> f:</C> | Ajoute à la fin sans effacer |
| Lire tout | <C>contenu = f.<F>read</F>()</C> | Retourne une seule chaîne (string) |
| Lire en liste | <C>lignes = f.<F>readlines</F>()</C> | Retourne une liste de strings (avec `\n`) |
| Lire ligne par ligne | <C><K>for</K> ligne <K>in</K> f:</C> | Itère sur chaque ligne |
| Retirer les sauts de ligne | <C>ligne.<F>strip</F>()</C> | Retire `\n` en début et fin |
| Écrire | <C>f.<F>write</F>(<S>"texte\n"</S>)</C> | Écrit une chaîne dans le fichier |
