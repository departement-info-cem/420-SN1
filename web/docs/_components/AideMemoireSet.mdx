Ensemble non ordonné et sans doublons, défini avec des accolades `{}`.

| Opération | Syntaxe | Description |
|---|---|---|
| Créer | <C>couleurs = {"{"}<S>"rouge"</S>, <S>"vert"</S>, <S>"bleu"</S>{"}"}</C> | Nouvel ensemble |
| Créer (vide) | <C>couleurs = <B>set</B>()</C> | Ensemble vide (pas <C>{"{}"}</C>) |
| Présence | <C>est_present = <S>"rouge"</S> <K>in</K> couleurs</C> | Retourne <C><K>True</K></C> ou <C><K>False</K></C> |
| Ajouter | <C>couleurs.<F>add</F>(<S>"jaune"</S>)</C> | Ajoute un élément (sans doublon) |
| Supprimer (erreur si absent) | <C>couleurs.<F>remove</F>(<S>"vert"</S>)</C> | Lève une erreur si absent |
| Supprimer (sans erreur) | <C>couleurs.<F>discard</F>(<S>"vert"</S>)</C> | Ne fait rien si absent |
| Union | <C>ensemble_union = a &#124; b</C> | Tous les éléments des deux ensembles |
| Intersection | <C>ensemble_commun = a & b</C> | Éléments communs aux deux ensembles |
| Différence | <C>ensemble_diff = a - b</C> | Éléments dans <C>a</C> mais pas dans <C>b</C> |
| Convertir en liste | <C>ma_liste = <B>list</B>(couleurs)</C> | Transforme l'ensemble en liste |
| Longueur | <C>nb_elements = <B>len</B>(couleurs)</C> | Nombre d'éléments |
