| Opération | Syntaxe | Description |
|---|---|---|
| Créer | `heros = ["Mario", "Link"]` | Nouvelle liste |
| Accéder | `heros[0]` | Élément à l'index 0 |
| Modifier | `heros[0] = "Samus"` | Remplacer un élément |
| Longueur | `len(heros)` | Nombre d'éléments |
| Présence | `"Mario" in heros` | Retourne `True` ou `False` |
| Ajouter (fin) | `heros.append("Pikachu")` | Ajoute à la fin |
| Insérer | `heros.insert(1, "Samus")` | Insère à l'index 1 |
| Supprimer (valeur) | `heros.remove("Samus")` | Supprime la 1re occurrence |
| Supprimer (dernier) | `heros.pop()` | Supprime le dernier élément |
| Supprimer (premier) | `heros.pop(0)` | Supprime le premier élément |
| Trier (nouvelle liste) | `sorted(heros)` | Retourne une copie triée |
| Trier (en place) | `heros.sort()` | Modifie la liste directement |
| Combiner | `liste1 + liste2` | Concatène deux listes |
| Copier | `heros.copy()` | Copie indépendante |
| Maximum | `max([10, 42, 7])` | Plus grande valeur (nombres seulement) |
| Minimum | `min([10, 42, 7])` | Plus petite valeur (nombres seulement) |
| Somme | `sum([10, 42, 7])` | Somme de tous les éléments (nombres seulement) |
