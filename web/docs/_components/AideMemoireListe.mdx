Liste ordonnée et modifiable, définie avec des crochets `[]`.

| Opération | Syntaxe | Description |
|---|---|---|
| Créer | <C>heros = [<S>"Mario"</S>, <S>"Link"</S>]</C> | Nouvelle liste |
| Accéder | <C>heros[<N>0</N>]</C> | Élément à l'index 0 |
| Modifier | <C>heros[<N>0</N>] = <S>"Samus"</S></C> | Remplacer un élément |
| Longueur | <C><B>len</B>(heros)</C> | Nombre d'éléments |
| Présence | <C><S>"Mario"</S> <K>in</K> heros</C> | Retourne <C><K>True</K></C> ou <C><K>False</K></C> |
| Ajouter (fin) | <C>heros.<F>append</F>(<S>"Pikachu"</S>)</C> | Ajoute à la fin |
| Insérer | <C>heros.<F>insert</F>(<N>1</N>, <S>"Samus"</S>)</C> | Insère à l'index 1 |
| Supprimer (valeur) | <C>heros.<F>remove</F>(<S>"Samus"</S>)</C> | Supprime la 1re occurrence |
| Supprimer (dernier) | <C>heros.<F>pop</F>()</C> | Supprime le dernier élément |
| Supprimer (premier) | <C>heros.<F>pop</F>(<N>0</N>)</C> | Supprime le premier élément |
| Trier (nouvelle liste) | <C><B>sorted</B>(heros)</C> | Retourne une copie triée |
| Trier (en place) | <C>heros.<F>sort</F>()</C> | Modifie la liste directement |
| Combiner | <C>liste1 + liste2</C> | Concatène deux listes |
| Copier | <C>heros.<F>copy</F>()</C> | Copie indépendante |
| Maximum | <C><B>max</B>([<N>10</N>, <N>42</N>, <N>7</N>])</C> | Plus grande valeur (nombres seulement) |
| Minimum | <C><B>min</B>([<N>10</N>, <N>42</N>, <N>7</N>])</C> | Plus petite valeur (nombres seulement) |
| Somme | <C><B>sum</B>([<N>10</N>, <N>42</N>, <N>7</N>])</C> | Somme de tous les éléments (nombres seulement) |
