Sauvegarder et relire des structures Python (dictionnaires, listes) avec le module <C><K>import</K> json</C>, inclus avec Python.

| Opération | Syntaxe | Description |
|---|---|---|
| Importer | <C><K>import</K> json</C> | Module inclus, aucune installation requise |
| Sauvegarder | <C>json.<F>dump</F>(data, f, indent=<N>4</N>, ensure_ascii=<K>False</K>)</C> | Écrit un objet Python dans un fichier JSON |
| Charger | <C>data = json.<F>load</F>(f)</C> | Lit un objet Python à partir d'un fichier JSON |
