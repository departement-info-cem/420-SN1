Paires clé-valeur, défini avec des accolades `{}`.

| Opération | Syntaxe | Description |
|---|---|---|
| Créer | <C>experience = {"{"}<S>"nom"</S>: <S>"Électrolyse"</S>, <S>"tentative"</S>: <N>2</N>{"}"}</C> | Nouveau dictionnaire |
| Créer (vide) | <C>experience = {"{}"}</C> | Dictionnaire vide |
| Accéder | <C>experience[<S>"nom"</S>]</C> | Valeur associée à la clé |
| Ajouter / Modifier | <C>experience[<S>"succes"</S>] = <K>True</K></C> | Ajoute ou écrase la clé |
| Supprimer | <C><K>del</K> experience[<S>"tentative"</S>]</C> | Supprime la paire clé-valeur |
| Présence | <C><S>"nom"</S> <K>in</K> experience</C> | Vérifie si la clé existe |
| Clés | <C>experience.<F>keys</F>()</C> | Retourne toutes les clés |
| Valeurs | <C>experience.<F>values</F>()</C> | Retourne toutes les valeurs |
| Paires | <C>experience.<F>items</F>()</C> | Retourne les paires (clé, valeur) |
| Longueur | <C><B>len</B>(experience)</C> | Nombre de paires clé-valeur |
| Itérer (clés) | <C><K>for</K> cle <K>in</K> experience.<F>keys</F>():</C> | Parcourir toutes les clés |
| Itérer (valeurs) | <C><K>for</K> val <K>in</K> experience.<F>values</F>():</C> | Parcourir toutes les valeurs |
| Itérer (paires) | <C><K>for</K> cle, val <K>in</K> experience.<F>items</F>():</C> | Parcourir clés et valeurs |
