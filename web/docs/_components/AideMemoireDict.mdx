| Opération | Syntaxe | Description |
|---|---|---|
| Créer | `personne = {"nom": "Alice", "âge": 30}` | Nouveau dictionnaire |
| Créer (vide) | `personne = {}` | Dictionnaire vide |
| Accéder | `personne["nom"]` | Valeur associée à la clé |
| Ajouter / Modifier | `personne["ville"] = "Montréal"` | Ajoute ou écrase la clé |
| Supprimer | `del personne["âge"]` | Supprime la paire clé-valeur |
| Présence | `"nom" in personne` | Vérifie si la clé existe |
| Clés | `personne.keys()` | Retourne toutes les clés |
| Valeurs | `personne.values()` | Retourne toutes les valeurs |
| Paires | `personne.items()` | Retourne les paires (clé, valeur) |
| Longueur | `len(personne)` | Nombre de paires clé-valeur |
| Itérer (clés) | `for cle in personne.keys():` | Parcourir toutes les clés |
| Itérer (valeurs) | `for val in personne.values():` | Parcourir toutes les valeurs |
| Itérer (paires) | `for cle, val in personne.items():` | Parcourir clés et valeurs |
