Liste ordonnée et immuable, définie avec des parenthèses `()`.

| Opération | Syntaxe | Description |
|---|---|---|
| Créer | <C>coordonnees = (<N>10</N>, <N>20</N>)</C> | Nouveau tuple |
| Créer (sans parenthèses) | <C>coordonnees = <N>10</N>, <N>20</N></C> | Équivalent avec parenthèses implicites |
| Convertir une liste | <C>mon_tuple = <B>tuple</B>(ma_liste)</C> | Transforme une liste en tuple |
| Accéder | <C>element = coordonnees[<N>0</N>]</C> | Élément à l'index 0 |
| Indice négatif | <C>dernier = coordonnees[-<N>1</N>]</C> | Dernier élément |
| Longueur | <C>nb_elements = <B>len</B>(coordonnees)</C> | Nombre d'éléments |
| Présence | <C>est_present = <N>10</N> <K>in</K> coordonnees</C> | Retourne <C><K>True</K></C> ou <C><K>False</K></C> |
| Décomposer | <C>x, y = coordonnees</C> | Assigne chaque élément à une variable |
| Retour multiple | <C><K>return</K> a, b</C> | Retourne plusieurs valeurs (tuple implicite) |
