Liste ordonnée et immuable, définie avec des parenthèses `()`.

| Opération | Syntaxe | Description |
|---|---|---|
| Créer | `coordonnees = (10, 20)` | Nouveau tuple |
| Créer (sans parenthèses) | `coordonnees = 10, 20` | Équivalent avec parenthèses implicites |
| Convertir une liste | `tuple(ma_liste)` | Transforme une liste en tuple |
| Accéder | `coordonnees[0]` | Élément à l'index 0 |
| Indice négatif | `coordonnees[-1]` | Dernier élément |
| Longueur | `len(coordonnees)` | Nombre d'éléments |
| Présence | `10 in coordonnees` | Retourne `True` ou `False` |
| Décomposer | `x, y = coordonnees` | Assigne chaque élément à une variable |
| Retour multiple | `return a, b` | Retourne plusieurs valeurs (tuple implicite) |
