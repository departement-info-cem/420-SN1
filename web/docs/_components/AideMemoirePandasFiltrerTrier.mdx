| Opération | Syntaxe | Description |
|---|---|---|
| Filtre simple (nombre) | <C>df_filtrer = df[df[<S>"tip"</S>] > <N>5</N>]</C> | Pourboire supérieur à 5 $ |
| Filtre simple (texte) | <C>df_filtrer = df[df[<S>"time"</S>] == <S>"Dinner"</S>]</C> | Égalité exacte sur une colonne texte |
| Filtre texte (.str.) | <C>df_filtrer = df[df[<S>"day"</S>].str.<F>contains</F>(<S>"Th"</S>)]</C> | Colonne contient une sous-chaîne |
| Filtre ET | <C>df_filtrer = df[(df[<S>"day"</S>] == <S>"Sun"</S>) & (df[<S>"time"</S>] == <S>"Dinner"</S>)]</C> | Les deux conditions vraies |
| Filtre OU | <C>df_filtrer = df[(df[<S>"size"</S>] == <N>1</N>) \| (df[<S>"size"</S>] == <N>2</N>)]</C> | Au moins une condition vraie |
| Filtre liste | <C>df_filtrer = df[df[<S>"day"</S>].<F>isin</F>([<S>"Thur"</S>, <S>"Fri"</S>])]</C> | Valeur parmi une liste |
| Trier croissant | <C>df_trier = df.<F>sort_values</F>(by=<S>"tip"</S>)</C> | Du plus petit au plus grand |
| Trier décroissant | <C>df_trier = df.<F>sort_values</F>(by=<S>"tip"</S>, ascending=<K>False</K>)</C> | Du plus grand au plus petit |
| Trier multi-colonnes | <C>df_trier = df.<F>sort_values</F>(by=[<S>"day"</S>, <S>"tip"</S>])</C> | Tri sur plusieurs colonnes |
| Chaîner filtre + tri | <C>df_resultat = df[df[<S>"day"</S>] == <S>"Sun"</S>].<F>sort_values</F>(by=<S>"tip"</S>)</C> | Filtrer puis trier en une ligne |

:::warning ⚠️ Ces opérations ne modifient pas `df`
`sort_values()` et les filtres retournent toujours un nouveau DataFrame. `df` original reste intact.
:::
