        :::info Compr√©hension de liste (List Comprehension)

        La compr√©hension de liste est une mani√®re concise et √©l√©gante de cr√©er des listes en Python.         La syntaxe g√©n√©rale de la compr√©hension de liste est la suivante :
        ```python showLineNumbers

        ```
        - `expression` : l'op√©ration ou la transformation √† appliquer √† chaque √©l√©ment.
        - `element` : la variable qui prend chaque valeur de la s√©quence.
        - `sequence` : la s√©quence (comme une liste) que l'on parcourt.
        - `condition` (optionnelle) : une condition pour filtrer les √©l√©ments.

        Voici un exemple simple pour cr√©er une liste des carr√©s des nombres pairs d'une liste initiale :
        ```python showLineNumbers

        print(carr√©s_pairs)  # Affiche [4, 16, 36]
        ```
        Dans cet exemple :
        - Nous parcourons chaque nombre `x` dans la liste `nombres`.
        - Nous calculons le carr√© de `x` (`x ** 2`) uniquement si `x` est pair (`if x % 2 == 0`).
        - Le r√©sultat est une nouvelle liste `carr√©s_pairs` contenant les carr√©s des nombres pairs.

        Ex√©cutez ce code en mode d√©bogage pour bien comprendre son fonctionnement.<br />
        Mettez un point d'arr√™t sur la ligne 2 et avancez une ligne √† la fois pour suivre l'√©volution de la variable `x` et de la liste `carr√©s_pairs`.

        :::

        ---

        :::tip Avantages de la compr√©hension de liste
        - **Concision** : La compr√©hension de liste permet d'√©crire moins de lignes de code pour accomplir la m√™me t√¢che.
        - **Lisibilit√©** : Elle rend le code plus lisible en exprimant clairement l'intention de cr√©er une nouvelle liste bas√©e sur une transformation ou un filtrage.
        - **Performance** : La compr√©hension de liste est souvent plus rapide que l'utilisation de boucles `for` traditionnelles pour cr√©er des listes.
        :::
