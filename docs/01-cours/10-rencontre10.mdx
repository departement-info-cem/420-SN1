        :::tip Introduction

        Vous connaissez maintenant les 4 types de donn√©es de base en Python :<br />
        - les entiers (`int`) : 5, -3, 42, 0, ...
        - les nombres √† virgule (`float`) : 3.14, -0.001, 2.0, ...
        - les cha√Ænes de caract√®res (`str`) : "Bonjour", 'a', "123", ...
        - les bool√©ens (`bool`) : `True`, `False`

        Et deux structures de donn√©es principales :
        - les listes (`list`) : une collection ordonn√©e et modifiable d'√©l√©ments, d√©finie en utilisant des crochets `[]`.
        - les ensembles (`set`) : une collection non ordonn√©e et non index√©e d'√©l√©ments uniques, d√©finie en utilisant des accolades `{}`.

        Nous allons maintenant d√©couvrir deux autre types de [structures de donn√©es](../aidememoire/structuresdonnes) en Python :
        - les tuples (`tuple`).
        - les dictionnaires (`dict`).

        :::

        ---

        :::note Tuple

        Un tuple est une collection ordonn√©e et immuable d'√©l√©ments.<br/>
        Contrairement aux listes, les tuples ne peuvent pas √™tre modifi√©s apr√®s leur cr√©ation.<br/>
        Ils sont d√©finis en utilisant des parenth√®ses `()`.

        :::

        ---

        :::info Cr√©ation d'un tuple

        ```python
        # Cr√©ation d'un tuple

        ```

        On peut aussi cr√©er un tuple sans parenth√®ses :

        ```python

        ```

        On peut convertir une liste en tuple en utilisant la fonction `tuple()` :

        ```python

        mon_tuple = tuple(ma_liste)

        ```

        :::

        ---

        :::info Acc√®s aux √©l√©ments d'un tuple
        On peut acc√©der aux √©l√©ments d'un tuple en utilisant des indices, de la m√™me mani√®re que pour les listes :

        ```python

        print(mon_tuple[2])  # Output: 30
        ```

        Comme les listes, les indices n√©gatifs fonctionnent aussi :

        ```python

        print(mon_tuple[-3])  # Output: 30
        ```

        Cependant, comme les tuples sont immuables (contrairement aux listes), on ne peut pas modifier leurs √©l√©ments :

        ```python

        ```
        :::

        ---

        :::tip Retour de fonctions multiples

        Une fonction qui retourne plusieurs valeurs utilise souvent un tuple pour les regrouper.
        - Les parenth√®ses sont souvent omises apr√®s le `return`.
        - Les valeurs retourn√©es peuvent √™tre soit captur√©es dans une seule variable (un tuple), soit d√©compos√©es en plusieurs variables, ce qui est tr√®s pratique.

        ```python

        print(somme)    # Output: 8
        print(produit)  # Output: 15
        ```

        :::info Intro

        Le dictionnaire est une structure de donn√©es tr√®s puissante en Python, mais aussi plus complexe que les listes, les ensembles ou les tuples.

        Il ressemble √† un ensemble, o√π chaque √©l√©ment unique est en fait une cl√©, associ√©e √† une valeur.<br />
        Un peu comme un dictionnaire papier, o√π chaque mot (la cl√©) est associ√© √† une d√©finition (la valeur).<br />
        On peut acc√©der rapidement √† une valeur en connaissant sa cl√©.

        Par exemple, on peut cr√©er un dictionnaire pour stocker des informations sur une personne :
        - la cl√© "nom" est associ√©e √† la valeur "Alice"
        - la cl√© "√¢ge" est associ√©e √† la valeur 30
        - la cl√© "ville" est associ√©e √† la valeur "Montr√©al"

        Pour cr√©er un dictionnaire, on utilise des accolades `{}` et on s√©pare les cl√©s des valeurs avec des deux-points `:`.<br />
        Chaque paire cl√©-valeur est s√©par√©e par une virgule `,`.

        ```python

            "√¢ge": 30,

        ```

        On peut aussi cr√©er un dictionnaire vide et y ajouter des √©l√©ments plus tard (ce qui est plus courant et tr√®s pratique) :

        ```python

        mon_dict["√¢ge"] = 30
        ```

        :::

        ---

        :::info Acc√®s aux √©l√©ments d'un dictionnaire
        On peut acc√©der aux valeurs d'un dictionnaire en utilisant leurs cl√©s :


        On peut aussi modifier les valeurs associ√©es √† une cl√© :

        ```python

        print(mon_dict["√¢ge"])  # Output: 31
        ```

        :::

        ---

        :::info Types de cl√©s et valeurs
        - Les **cl√©s** dans un dictionnaire doivent √™tre de types immuables (comme les cha√Ænes de caract√®res, les nombres ou les tuples immuables).
        - Les **valeurs** peuvent √™tre de n'importe quel type, y compris des listes, des ensembles, des tuples ou m√™me d'autres dictionnaires.
        ```python

                "rue": "123 Rue Principale",
                "ville": "Montr√©al"

        }
        ```

        :::

        ---

        :::info It√©ration sur un dictionnaire
        On peut parcourir les cl√©s ou les paires cl√©-valeur d'un dictionnaire en utilisant une boucle `for` :

        L'ex√©cution de ce code affichera :

        :::

        ---

        :::info M√©thodes utiles des dictionnaires
        Voici quelques m√©thodes couramment utilis√©es avec les dictionnaires :

        - `mon_dict.values()` : retourne une `list` des valeurs du dictionnaire.
        - `mon_dict.items()` : retourne une `list` de tuples (cl√©, valeur).

        Exemple :
        ```python

        print(mon_dict.items())   # Output: dict_items([('nom', 'Alice'), ('√¢ge', 30), ('ville', 'Montr√©al')])
        ```

        :::

        ---

        :::tip Utilisations courantes des dictionnaires
        Les dictionnaires sont tr√®s utiles pour :
        - Stocker des donn√©es structur√©es (comme des informations sur une personne, un produit, etc.).
        - Compter des occurrences (par exemple, compter combien de fois chaque mot appara√Æt dans un texte).

        :::info Fichier texte

        Un fichier texte est un type de fichier tr√®s courant et simple √† manipuler.
        - Il est simplement constitu√© d'une suite de caract√®res lisibles par l'humain.
        - Il ne contient aucun formatage, juste du texte brut.
        - ‚úÖ Les scripts Python (`.py`) que vous √©crivez sont aussi des fichiers texte.

        ---

        :::info Structure d'un fichier texte

        Un fichier texte est organis√© en lignes.
        - Chaque ligne peut contenir plusieurs morceaux de donn√©es s√©par√©s par des espaces, des tabulations, des virgules, etc.
        - Les lignes sont s√©par√©es par des caract√®res sp√©ciaux appel√©s "sauts de ligne" (`\n`).

        On peut souvent ouvrir ou cr√©er un fichier texte
        - √† la main avec un √©diteur de texte (Notepad, Pycharm, VSCode, ...)
        - avec du code Python pour les manipuler

        Exemple de code Python pour lire un fichier texte ligne par ligne :

        ```python
        # Ouvrir un fichier en mode lecture

            # Lire chaque ligne du fichier
            for ligne in f:
                print(ligne, end="") # end="" pour √©viter d'ajouter des lignes vides suppl√©mentaires
        ```

        Exemple de code Python pour √©crire dans un fichier texte :

        ```python

        with open("monfichier.txt", "w") as f:
            # √âcrire des lignes dans le fichier
            f.write("Bonjour le monde!\n")
            f.write("Ceci est un fichier texte.\n")
        ```

        :::

        ---

        :::warning Gestion des fichiers avec le mot-cl√© with
        En Python, il est recommand√© d'utiliser le mot-cl√© `with` pour g√©rer les fichiers.
        - Cela garantit que le fichier est correctement ferm√© apr√®s son utilisation, m√™me en cas d'erreur.
        - Le fichier est automatiquement ferm√© lorsque le bloc `with` est quitt√©.
        ```python

            contenu = f.read()
        # Le fichier est automatiquement ferm√© ici
        ```
        :::

        ---

        :::info Autres fa√ßons de lire un fichier texte
        On peut aussi lire tout le contenu d'un fichier texte en une seule fois avec la m√©thode `read()` :

        La variable `lignes` contiendra alors une liste de cha√Ænes de caract√®res, chaque cha√Æne repr√©sentant une ligne du fichier, avec les sauts de ligne (`\n`) inclus :

        :::

        ---

        :::info Autres fa√ßons d'√©crire dans un fichier texte
