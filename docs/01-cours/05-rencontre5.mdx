        :::info Instructions `break`, `continue` et `pass`

        Parfois, on peut vouloir interrompre une boucle avant qu'elle ne se termine naturellement, ou sauter certaines it√©rations.<br />
        Python fournit trois instructions sp√©ciales pour g√©rer ces situations : `break`, `continue` et `pass`.

        ---

        :::info `break`
        Cette instruction permet de sortir imm√©diatement d'une boucle, qu'elle soit `for` ou `while`.<br />
        Lorsqu'un `break` est rencontr√©, l'ex√©cution du programme continue √† la ligne suivant la boucle.
        ```python showLineNumbers

        ```
        Dans cet exemple, la boucle s'arr√™te d√®s que `i` atteint 5.

        :::

        ---

        :::info `continue`
        Cette instruction permet de sauter le reste du code dans la boucle pour l'it√©ration en cours et de passer directement √† l'it√©ration suivante.

        Ici, seuls les nombres impairs sont affich√©s, car les nombres pairs sont ignor√©s gr√¢ce √† `continue`.

        :::

        ---

        :::info `pass`
        Cette instruction est un "no-op" (no operation), c'est-√†-dire qu'elle ne fait rien.<br />
        Elle est souvent utilis√©e comme un espace r√©serv√© dans des blocs de code o√π une instruction est requise syntaxiquement mais o√π aucune action n'est n√©cessaire.
        ```python showLineNumbers

        ```
        Dans cet exemple, rien ne se passe pour les valeurs de `i` inf√©rieures √† 3, mais pour les autres valeurs, elles sont affich√©es.

        :::
