# Installer PyCharm

Pour installer PyCharm à la maison, voici la recette

1. Aller sur le site [PyCharm](https://www.jetbrains.com/pycharm/download/)
2. Descendre dans la page pour trouver la version Community
3. Cliquer sur le lien de téléchargement
4. Suivre les instructions d'installation

## Test sur un projet

1. Lance PyCharm
2. Crée un nouveau projet
3. Crée un nouveau fichier Python
4. Écris un petit programme avec
```python
print("Hello world")
```
5. Exécute le programme

## Si ça ne marche pas

Si ça ne marche pas, si c'est tu as un portable,
tu peux l'apporter en classe et on va regarder avec tron prof.
