# 🐍 Fichier Python

Les fichiers **Python** (`.py`) permettent d’écrire du code exécutable dans le langage Python.

Ils sont utilisés pour créer des scripts, des modules, des programmes ou des applications complètes.

---

## Créer un fichier Python dans PyCharm

1. Ouvrir un projet dans PyCharm (ou en [créer un nouveau](creer-projet)).
2. Dans l’arborescence à gauche, cliquer-droit sur le dossier où créer le fichier.
3. Choisir **New > Python File**.
4. Donner un nom au fichier (sans espace, sans accent), par exemple `exemple`, puis appuyer sur ⏎.
5. PyCharm crée automatiquement un fichier nommé `exemple.py` et l’ouvre dans un nouvel onglet.

---

## Écrire du code Python

1. Taper du code Python dans le fichier `.py` ouvert.

#### Exemple de code Python :

```python showLineNumbers
def saluer(nom):
    print(f"Bonjour, {nom}!")

saluer("Marie")
```

---

## Exécuter le code en mode normal

1. Cliquer-droit sur le nom du fichier dans l’arborescence à gauche.
2. Dans le menu contextuel, cliquer sur le bouton vert ▶ **Run**.
3. Pour exécuter le fichier de nouveau, en haut à droite de l’écran,
si le fichier est bien sélectionné, il suffit de cliquer sur le bouton vert ▶.
4. Le résultat s’affiche dans la console en bas de l’écran.

#### Résultat affiché :

```
Bonjour, Marie!
```

---

## Exécuter le code en mode *debug* 🪲

1. Cliquer-droit sur le nom du fichier dans l’arborescence à gauche.
2. Dans le menu contextuel, cliquer sur le bouton vert 🪲 **Debug**.
3. Pour exécuter le fichier de nouveau, en haut à droite de l’écran,
si le fichier est bien sélectionné, il suffit de cliquer sur le bouton vert 🪲.
4. Pour plus de détails sur l'utilisation du débogueur, voir la recette [Déboguer](deboguer-point-arret).

---

## Astuces

- Ne pas oublier d’indenter correctement le code (indentation = 4 espaces ou 1 tabulation).
